# Tocken-Bucket Rate Limiter

* Микросервис контроля запросов реализованный по алгоритму Tocken Bucket. Выдерживает сильные всплески запросов.
* Ограничитель запросов работает в оперативной памяти. Это позволяет обравабывать запросы на огромной скорости, и не иметь ограничений операций.

## Стек
* **Язык:** Go - эффективная обработка конкурентных запросов.
* **Хранилище:** In-Memory - для хранения состояния бакетов в оперативной памяти и распределеннию.
* **Контейнеризация:** Docker, Docker Compose - для быстрого развертывания.
* **Автоматизация:** GitHub Actions (CI/CD) - автоматический запуск unit-тестов при каждом пуш.

## Возможности
* **Динамическая настройка параметров** - параметры передаются в запросе, что позволяет легко настраивать каждый запрос.
* **Многопоточность** - безопасная работа с конкурентностью.
